# SwiftBoard - Vanilla Kanban

A simple drag-and-drop Kanban board built with plain HTML, CSS, and JavaScript.

## Features
- Drag and drop tasks between columns (To Do, In Progress, Done)
- Simple, clean interface

## Usage
To run the application, you can serve the directory using a static file server. For example, with Python:

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000 in your browser.
